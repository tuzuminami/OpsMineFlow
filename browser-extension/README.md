# browser-extension

Browser collection is planned after the native agent preview as a separate, optional installation. It must use optional domain permissions, remain local-only, show enabled/paused state, and mask URL paths and page titles before local ingestion.

Default `<all_urls>` access, page content, form values, input events, cookies, authentication data, browsing history, downloads, screenshots, audio, and camera access are prohibited.

See [docs/product/COLLECTION_ROADMAP.md](../docs/product/COLLECTION_ROADMAP.md).
