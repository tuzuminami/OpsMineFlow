# mac-agent

The first native macOS collection preview is planned as Swift-only and remains disabled by default. It will collect only explicitly approved app/window metadata with visible start, pause, and stop controls.

No separate helper is approved for the first release. Keystrokes, passwords, input text, clipboard contents, screenshots, recordings, microphone input, camera input, and hidden collection are prohibited.

See [docs/product/COLLECTION_ROADMAP.md](../docs/product/COLLECTION_ROADMAP.md).
