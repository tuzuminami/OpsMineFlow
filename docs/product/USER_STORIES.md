# User Stories

- As a consultant, I can import CSV logs so that I can start analysis without installing a monitoring agent.
- As a consultant, I can import ActivityWatch-style JSON so that existing local exports can be reused.
- As a participant, I can understand what data is collected so that I can consent safely.
- As a security reviewer, I can verify that runtime traffic is local-only.
- As a consultant, I can export draw.io diagrams so that process maps can be edited in client workshops.
- As a consultant, I can generate a Markdown report draft so that interview preparation is faster.
