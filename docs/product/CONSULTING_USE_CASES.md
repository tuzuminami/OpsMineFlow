# Consulting Use Cases

## Scenario 1: BPR As-Is Discovery

1. Explain the tool and data scope.
2. Obtain consent.
3. Collect or receive one week of local exports.
4. Import CSV, JSON, or optional ActivityWatch export.
5. Apply business labels.
6. Generate a process map.
7. Export a draw.io flow diagram.
8. Draft hypotheses before interviews.
9. Run follow-up interviews.
10. Organize issues.
11. Design To-Be options.

## Scenario 2: RPA and Automation Candidate Discovery

1. Extract repeated work.
2. Detect app round trips.
3. Detect rule-based tasks.
4. Score automation candidates.
5. Classify candidates as RPA, AI adoption review, system change, or operating rule change.
6. Estimate improvement impact.

## Scenario 3: Core System Renewal Discovery

1. Import business logs or CSV exports.
2. Visualize actual process flows.
3. Extract exception variants.
4. Detect manual work and external spreadsheet use.
5. Export draw.io diagrams.
6. Build requirements discussion points.
