# License Decision

OpsMineFlow uses Apache-2.0 for the project license.

## Rationale

- Friendly to commercial consulting delivery.
- Easier to explain to client legal and security teams.
- Permissive redistribution.
- Includes an explicit patent license.
- Avoids strong copyleft obligations that are hard to manage in client environments.

## Dependency Direction

Direct dependencies should be MIT, Apache-2.0, BSD-2-Clause, BSD-3-Clause, ISC, Zlib, Unlicense, CC0, or PSF License.

MPL-2.0 projects may be used only as optional external tools or import formats unless a separate legal review approves a narrow use.
