# Mining Pipeline

1. Load CSV or JSON.
2. Normalize into the standard event schema.
3. Mask sensitive fields.
4. Apply exclusion rules.
5. Assign business labels.
6. Sessionize events.
7. Calculate duration metrics.
8. Detect app switches and round trips.
9. Detect repeated patterns and transfer risk.
10. Build Directly-Follows Graph.
11. Analyze variants.
12. Detect bottlenecks.
13. Score automation candidates.
14. Export reports and diagrams.

The implementation avoids PM4Py and uses local algorithms.
