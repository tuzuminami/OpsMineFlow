# Privacy and Security

OpsMineFlow is designed for consent-based business improvement analysis.

## Privacy Controls

- URL path masking.
- Window-title masking.
- Domain-only setting.
- Excluded apps.
- Excluded domains.
- Excluded keywords.
- Local deletion.
- Retention settings.
- Anonymous user IDs.
- Export preview and warning.

## Security Controls

- Local files only.
- Localhost-only API bind.
- Restricted CORS.
- No external telemetry.
- No LLM integrations.
- Audit-friendly documentation.

## Data Not Collected

- Keystrokes.
- Passwords.
- Input text.
- Continuous clipboard text.
- Screenshots.
- Screen recordings.
- Audio.
- Camera images.
